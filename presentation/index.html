Slide 6 | Integration Highlight

Intercom API Integration Core

Integration Flow

  • User submits feedback
  • Backend validates input
  • Backend finds or creates Intercom contact
  • Backend creates conversation
  • Admin replies through dashboard

Challenges and Solutions

  • API failures → Retry logic
  • Missing contacts → Contact search fallback
  • Message conflicts → Better backend logging

Slide 8 | Hero Code

Example Intercom API Request

await axios.post(
  `${INTERCOM_BASE_URL}/conversations`,
  {
    from: {
      type: "contact",
      id: contactId
    },
    body: message
  },
  {
    headers: {
      Authorization: `Bearer ${INTERCOM_ACCESS_TOKEN}`,
      "Content-Type": "application/json"
    }
  }
);

Backend sends user message to Intercom
Intercom creates a conversation
Admin can reply through dashboard
